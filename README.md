# Sipeng Xie Personal Site

Static personal academic homepage generated from the local resume.

## GitHub Pages Deployment

1. Create a repository named `<your-github-username>.github.io`.
2. Copy the contents of this `personal-site` directory to the repository root.
3. Commit and push to the `main` branch.
4. Open `https://<your-github-username>.github.io`.

The page is static and does not require a build step.
